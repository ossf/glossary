a. The specific path or method a threat actor uses to exploit a vulnerability. Examples include phishing emails, a compromised API endpoint, or an exposed server port. Attack Vector.

b. The general process by which neglect could result in unintentional negative outcomes. Examples include poorly scoped development tools, insufficient training, or insecure default settings.

Part of Gemara Layer 1.
